Machine learning

Input Features

For each location, the average number of daily violations over all historical data contained in the training set was included as an feature. Additionally, I also used daily-weather-related features, including highest temperature, lowest temperature, rain/snow precipitation, accumulated snow depth on the ground.

Model: Random Forests

Since the location attribute appears strongly influencing the output, I use the mean model results as the benchmark. Then I applied the random forest regressor, which represents an ensemble method for regression machine learning problems. The method generally improves the performance of the prediction.

Cross validation

The data set is split into the training and test data sets, which contain 80% and 20% of the whole data. I applied cross validation to measure the prediction error.

Accuracy

Comparing with the mean model, The mean-squared-error (MSE) values of the random forest method are 89% and 85% for red light and speeding violations, respectively.
